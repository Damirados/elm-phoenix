# elm-phoenix

Elm adapter for Phoenix channels

## Example

Here's how to launch an Elm version of the standard Phoenix chat example:

* `git clone https://github.com/voltone/phoenix_chat_example.git`
* `git checkout elm`
* `git submodule init`
* `git submodule update`
* `mix deps.get`
* `npm install`
* `mix phoenix.server`
